# 5강. 기본 연산자
## 종류
- 덧셈, 뺄셈, 곱셈, 나눗셈, 나머지(%)
## 나머지(%)를 어디에 쓸까?
- 홀수 판별 : X % 2 = 1
- 짝수 판별 : Y % 2 = 0
- 0 ~ n 사이 값만 반환 : X % (n+1)
---
## 거듭제곱
- ** 이라는 연산자를 사용한다. `const num = 2 ** 3; // 결과 8`
---
## 우선순위
- 모를수가 없어서 패스
---
## 연산자 축약형
- a += b : a = a + b 
- a -= b : a = a - b 
- a *= b : a = a * b 
- a /= b : a = a / b 
- a %= b : a = a % b 
---
## 증감 연산자
- a++ : 지금은 a지만 다음줄은 a+1이 된다.
- a-- : 지금은 a지만 다음줄은 a-1이 된다.
- ++a : 지금부터 a+1이다.
- --a : 지금부터 a-1이다.
