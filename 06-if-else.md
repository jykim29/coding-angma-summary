# 6강. 비교 연산자와 조건문(if, else)
## 비교 연산자
- 부등호 (> , <, >=, <=)
---
## 동등 연산자
- equal (==, ===, !==, !===)
- ==는 숫자만 비교, ===는 숫자+타입까지 비교
- boolean으로 반환
---
## 조건문
- if문 : if(조건) { ... } 조건이 참일 경우 작동
- else if : if(조건)이 거짓일 경우 또 다른 조건 추가하여 else if(조건) { ... } 사용
- else : if와 else if 모두 거짓일 경우 모든 나머지 경우에 대하여 else { ... } 사용
